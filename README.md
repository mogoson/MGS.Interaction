# MGS.Interaction

## Summary

- Interaction component for Unity development.

## Ability

- Interact with Condition, Constraint and Transition.
- Interact with Collider, Mouse and Pointer.
- Interact with Clickable, Grabbable, Placeable and Touchable.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.Interaction.git?path=/Assets
  ```

## Samples

- Unity --> Window --> Package Manager --> Packages-Mogoson --> Interaction --> Samples.
